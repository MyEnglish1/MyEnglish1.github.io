list
Список
Список (list) — это упорядоченный набор элементов, каждый из которых имеет свой номер, или индекс, позволяющий быстро получить к нему доступ. Нумерация элементов в списке начинается с 0
Как создать list
Как создать список
Чтобы создать объект list, в Python используют квадратные скобки — [].
a = [1, 2, 3]
Мы создали список a и поместили в него три числа, которые разделили запятыми. Давайте выведем его с помощью функции print():
print(a) >>> [1, 2, 3]
Индексация
Индексация
Доступ к элементам списка получают по индексам, через квадратные скобки []:
a = [1, 2, 3]
print(a[1]) >>> 2 Мы обратились ко второму элементу и вывели его с помощью print(). Здесь важно помнить две вещи: у каждого элемента есть свой индекс; индексы начинаются с 0.
Объединение списков
Объединение списков
a = [1, 2]
b = [3, 4]
с = a + b
print(с) >>> [1, 2, 3, 4]
Сравнение списков
Сравнение списков
a = [1, 2, 3]
b = [1, 2, 3]
print(a == b) >>> True
len()
len()
Возвращает длину списка:
a = [5, 3, 1]
len(a) # 3
sorted()
sorted()
Возвращает отсортированный список:
a = [8, 1, 3, 2]
sorted(a) # [1, 2, 3, 8]
min() и max()
min() и max()
Возвращают наименьший и наибольший элемент списка:
a = [1, 9, -2, 3]
min(a) # -2
max(a) # 9
append()
append()
Добавляет новый элемент в конец списка:
a = [1, 2, 3]
a.append(4)
print(a)
>>> [1, 2, 3, 4]
extend()
extend()
Добавляет набор элементов в конец списка:
a = [1, 2, 3]
a.extend([4, 5])
print(a)
>>> [1, 2, 3, 4, 5]
insert()
insert()
Добавляет новый элемент по индексу:
a = [1, 2, 3]
a.insert(0, 4)
print(a) >>> [4, 1, 2, 3]
Сначала мы передаём индекс, по которому хотим вставить новый элемент, а затем сам элемент.
remove()
remove()
Удаляет элемент из списка:
a = [1, 2, 3, 1]
a.remove(1)
print(a) >>> [2, 3, 1] Метод удаляет только первое вхождение элемента. Остальные остаются нетронутыми.
clear()
clear()
Удаляет все элементы из списка и делает его пустым:
a = [1, 2, 3]
a.clear()
print(a) >>> []
index()
index()
Возвращает индекс элемента списка в Python:
a = [1, 2, 3]
print(a.index(2)) >>> 1
pop()
pop()
Удаляет элемент по индексу

count()
count()
Считает, сколько раз элемент повторяется в списке:
a = [1, 1, 1, 2]
print(a.count(1)) >>> 3
sort()
sort()
Сортирует список:
a = [4, 1, 5, 2]
a.sort() # [1, 2, 4, 5]
copy()
copy()
Копирует список:
a = [1, 2, 3]
b = a.copy()
print(b) >>> [1, 2, 3]
